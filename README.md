## Características

- Actualización de datos en tiempo real.
- Passkey para iniciar con huella digital. (Usando LIN Vault) 
- Diseño responsive.
- Multi-Inventario. (Compartir inventarios con otras personas)


## Tecnologías

- .NET 8.
- Blazor.
- HTML, CSS y JavaScript.
- Tailwind CSS.
